Usage: db-auto <command> [options]

Arguments:
  path                         the list of table names joined by a . For
                               example driver.mission.mission_aud
  id                           the id of the primary key in the first table in
                               the path

Options:
   --plan                      show the plan instead of executing (default:
                               false)
  -s, --sql                    show the sql instead of executing (default:
                               false)
  --fullSql                    normally the show sql doesn't include limits.
                               This shows them (default: false)
  -c, --count                  returns the count of the items in the table
                               (default: false)
  -d, --distinct               only return distinct values (default: false)
  --page <page>                which page of the results should be returned
  --pageSize <pageSize>        Changes the page of the returned results
                               (default: 15)
  -t, --trace                  trace the results (default: false)
  -j, --json                   Sql output as json instead of columns (default:
                               false)
  --onelinejson                Sql output as 'one json per line for the row'
                               instead of columns (default: false)
  --notitles                   Sql output as columns doesn't have titles
                               (default: false)
  -w, --where [where...]       a where clause added to the query. There is no
                               syntax checking (default: [])
  -V, --version                output the version number
  --name <name>                driver name
  --employeeNum <employeeNum>
  --date <date>
  -h, --help                   display help for command

Commands:
  mocks [tables...]
  envs
  tables
