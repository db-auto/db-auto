Usage: db-auto <command> [options]

Arguments:
  path                         the list of table names joined by a . For
                               example driver.mission.mission_aud
  id                           the id of the primary key in the first table in
                               the path

Options:
  -p, --plan                   show the plan instead of executing (default:
                               false)
  -s, --sql                    show the sql instead of executing (default:
                               false)
  -t, --trace                  trace the results (default: false)
  -w, --where [where...]       a where clause added to the query. There is no
                               syntax checking (default: [])
  -V, --version                output the version number
  --name <name>                driver name
  --employeeNum <employeeNum>
  --date <date>
  -h, --help                   display help for command

Commands:
  mocks [tables...]
  envs
  tables
